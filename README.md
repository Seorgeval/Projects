# Projects
Global Application Builder (GAB) projects in Global Shop Solutions ERP software
  ## 7521
  A dashboard that displays all routers based on user-selected criteria. This allows users to mass update routers based on the first material sequence, if it contains a die color, or if it is for a specific customer.
  ## 7528 
  Simple script that adds a work order task sequence in the background when a user generates a work order from a sales order
  ## 7580
  A background script that runs when a user is staging shipments that will auto stage a secondary line as well if it was already linked to another line on the sales order.
  ## 7638
  A form that assists users with staging shipments by calculating inventory quantities to be staged by summing totals from inventory lots and bins.
  ## 7729
  A screen that allows for user input in order to generate an open work order BI report.
